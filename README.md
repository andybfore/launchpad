## Introduction
A little vanilla music launchpad to play different drum sets using mouse click or keyboard press.

## Sets
| Keyboard    | Sound         |
| ----------- | ------------- |
| <kbd>Q</kbd>| Hi hat close 1|
| <kbd>W</kbd>| Hi hat open   |
| <kbd>E</kbd>| Hi hat close 2|
| <kbd>A</kbd>| Snare         |
| <kbd>S</kbd>| Snare 2       |
| <kbd>D</kbd>| Clap          |
| <kbd><</kbd>| Kick          |
| <kbd>Z</kbd>| Kick 2        |
| <kbd>X</kbd>| Kick 3        |
